# MealMagnet 🍽️

A responsive recipe discovery web application that helps users search, explore, and discover thousands of delicious recipes from around the world. Whether you're looking for a quick breakfast, a fancy dinner, or exploring new cuisines, MealMagnet has you covered!

## 🌟 Features

### 🔍 Smart Recipe Search
- Search for recipes by name (e.g., "chicken", "pasta", "pizza")
- Instantly view all matching recipes in a beautiful grid layout
- Each search result displays recipe images, titles, and quick access buttons

### 🏷️ Browse by Meal Categories
- **Breakfast**: Start your day with delicious breakfast recipes
- **Lunch**: Explore seafood and fresh lunch options
- **Snacks**: Discover vegan and healthy snack ideas
- **Dinner**: Find pasta and dinner recipes for the evening

### 📋 Complete Recipe Details
- Full list of ingredients with measurements
- Step-by-step cooking instructions
- High-quality recipe images
- Easy-to-read ingredient layout

### 💻 Responsive Design
- Works seamlessly on desktop, tablet, and mobile devices
- Beautiful gradient background and modern UI
- Smooth animations and intuitive navigation
- Dark mode friendly color scheme

## 🎯 How to Use

### Searching for a Recipe
1. Type the name of any recipe in the search bar at the top (e.g., "chicken", "pasta")
2. Click the search button or press Enter
3. Browse through all matching recipes in the modal
4. Click "View Recipe" on any recipe to see full details
5. Scroll down to view ingredients and cooking instructions

### Browsing by Category
1. Click on any meal category button: **Breakfast**, **Lunch**, **Snacks**, or **Dinner**
2. A modal will open showing all recipes in that category
3. Use the search box within the modal to filter recipes further
4. Click "View Recipe" to view complete details
5. Click the X button to close and return to the main page

### Viewing Recipe Details
- **Recipe Title**: The name of the dish
- **Hero Image**: A beautiful photo of the prepared recipe
- **Instructions**: Detailed cooking steps and techniques
- **Ingredients List**: All ingredients with exact measurements needed

## 🛠️ Technical Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **API**: [TheMealDB API](https://www.themealdb.com/api.php) - Free recipe database
- **Styling**: Custom CSS with modern gradients and animations
- **Icons**: Font Awesome 6.7.2
- **Fonts**: Google Fonts (Poppins, Playfair Display)
- **Architecture**: Client-side JavaScript with async/await

## 🔗 API Integration

MealMagnet uses the **TheMealDB API**, a free meal database API that provides:
- Meal search by name
- Meal categories and filters
- Complete recipe details with ingredients and instructions
- High-quality meal images

**API Endpoints Used:**
- `/search.php?s=` - Search meals by name
- `/filter.php?c=` - Filter meals by category
- `/lookup.php?i=` - Get meal details by ID
